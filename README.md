# 李锦辉 AI 应用求职作品集

目标方向：AI应用搭建、AI应用实施、智能体搭建、Agent应用开发、电商AI应用。

## 项目一：直播电商智能客服 Agent

基于本人直播电商业务经验设计，覆盖客服规则问答、订单、物流、库存、退款、投诉转人工、多轮缺参追问和订单权限校验。

项目目录：`ecommerce-customer-service-agent/`

## 项目二：企业费用报销预审与审批协同 Agent

用于展示企业内部流程自动化、结构化信息提取、规则判断、RAG制度查询、模拟API、SQLite、人工审批和审计日志。

状态：待创建。

## 真实性说明

两个项目均为个人业务原型，使用模拟业务数据，未接入企业生产系统。项目重点展示业务流程拆解、Agent方案设计、AI辅助开发、测试和文档能力。
